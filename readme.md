## Setup

Install requirements in virtual environment:

``` bash
python -m venv .venv
source .venv/bin/activate
python -m pip install -r requirements.txt
```

Then run the setup script

``` bash
python cli init
```

Now you can run the benchmark:

``` bash
python cli run
```

When you are finished, cleanup with:

``` bash
python cli cleanup
```
